Periodic Table Game Rules

     To begin the game you need to click the “Let the fun begin” button. First, you will answer the question above the input box. Next, you must click the “Continue” button. Once the you click the continue button, an alert will pop up. If the alert is red, you answered the question wrong. If the alert is green, you answered the question right. If the alert is yellow, you did not type anything for the last question. You get one point for every right answer. Also, the element based on the question is revealed on the periodic table. Your progress will be lost if you exit or reload the page (version two is on the way to fix this). The game can continue on until all the elements in the periodic table are revealed!
