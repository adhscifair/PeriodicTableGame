Periodic Table Game Rules
Welcome to the Periodic Table Game! Not sure where to start? Well, this is how you use the website to the best of your ability!

First hover and click on an elemenet that you would like to answer questions about! (As shown below)


Then a modal should pop up. Enter the answer into the input box, and click submit. (As shown below)


The alerts on top would tell you wether you got it right, wrong, or just left it blank. (As shown below)


After you answered all the questions for one elemenet, a congradulatory message would've poped up! (As shown below)


If you clicked the "progress" tab, you would be able to see what you got right, wrong and what your score is. (As shown below)


Close the pop-up, and then you can keep clicking on different elements and answering different questions about them.

That's all! Hope you have fun playing this game!
